<p>
There is something in the air... All kinds of good stuff is happening in the Wicket community... Here's a recap of the most recent events:</p>
<ul>
<li>Wicket 1.3 released - <a href="http://www.apache.org/dyn/closer.cgi/wicket/1.3.0">download it now</a>!</li>
<li>Dan Syrstad released <a href="http://wicketwebbeans.sourceforge.net/">Wicket Web Beans 1.0</a>. <acronym title="Wicket Web Beans">WWB</acronym> is a component toolkit for displaying and editing POJOs that conform to the JavaBeans specification.</li>
<li>Matej Knopp has released <a href="http://www.inmethod.com/">TreeGrid and DataGrid</a> components as Apache 2.0 Licensed components</li>
<li>The <a href="http://jweekend.com/dev/LWUGReg/" title="London Wicket">London Wicket</a> event needs a bigger room</li>
<li>Maurice Marrink has created a first release candidate for his Wicket security projects <a href="#">Wasp and Swarm</a></li>
<li><a href="http://manning.com/dashorst">Wicket in Action</a> is nearing completion - chapter 15 is undergoing a final review round, and Eelco is working hard on completing chapter 14</li>
<li><a href="http://fabulously40.com">Fabulously40.com</a>, a social network for fabulous women over 40 was converted from Ruby on Rails to Wicket. The convert  <a href="http://digg.com/programming/Apache_Wicket_1_3_released?t=11740164#c11740164">writes</a>:<br />
<blockquote>I recently converted a Rails site to Wicket. Wicket really cuts down on template spaghetti code. I can honestly say that wicketâ€™s OO approach is the right way of doing things for web application development.</p></blockquote>
</li>
<li><a href="http://www.nabble.com/-ANNOUNCE--Apache-Portals-Jetspeed-2.1.3-release-td14540482.html#a14698311">Apache Jetspeed 2.1.3</a> provides out-of-the-box Wicket Portlet support</li>
<li>An interview with me in <a href="http://www.javalobby.org/java/forums/t105230.html">JavaLobby</a>, and in <a href="http://www.infoworld.com/article/08/01/03/apache-wicket_1.html">InfoWorld</a></li>
<li>Ryan Sonnek implemented improved <a href="http://www.jroller.com/wireframe/entry/revamped_drag_and_drop_for">drag and drop support for scriptaculous</a></li>
<li>Google charts has a <a href="http://www.codecommit.com/blog/java/a-wicket-api-for-google-charts">Wicket implementation</a></li>
<li>CSS drop down menus (Suckerfish) available as a <a href="http://javathoughts.capesugarbird.com/2007/11/suckerfish-dropdowns-for-wicket.html">wicket stuff project</a></li>
</ul>
<p>And we've just started 2008, so this will be a big year!</p>
